# library-python-cli

Library management system based in Python as a command-line application. Practices OOP principles to manage books, users, and checkouts efficiently.

The Library management system introduces an interface to perform library tasks. We can add books, users, display available books, borrow or return books, print borrowed books, and print users. The book data is stored in json to easily aggregate and build the book and user data.

Example Usage:

```
Program Options:
 1) Add book
 2) Add user
 3) Print available books
 4) Print books by prefix
 5) Borrow book
 6) Return book
 7) Print users borrowed book
 8) Print users

Enter your choice from 1 to 8: 1

Please Enter the book id: 100
Please Enter the book name: The Lord of the Rings
Please Enter the book quantity: 3

Book is added successfully!

(Continue using the menu to manage your library...)
```
